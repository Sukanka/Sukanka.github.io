前言

我想要在 archlinux 上装一个 windows 虚拟机，并且希望虚拟机性能比较好。以前尝试过 vmware， 但是感觉图形性能不行。因此考虑了qemu。本文主要涉及以下几点：

  1. cpu: host-passthrough
  2. gpu: 透传
  3. virtio

准备工作

启用虚拟化

  1. 首先在 BIOS 里面开启虚拟化支持，这个中文资料很多，也不难，这里不叙述了。

  2. 启用 Linux 系统的虚拟化参数

  3. 安装 qemu 及图形界面

yay -S qemu virt-manager
